Set up Claude Code to work with both the Outfox MCP server and this Mintlify
docs repo.

## Prerequisites

- Active Claude subscription (Pro, Max, or API access)
- Access to an Outfox account with organization membership

## Setup

<Steps>
<Step title="Install Claude Code">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Connect the Outfox MCP server">
  Add the remote MCP server URL used by Outfox:

  ```bash
  https://app.outfox.ai/api/mcp
  ```

  Claude Code will handle the OAuth flow and prompt you to sign in with Clerk.
</Step>

<Step title="Install the Mintlify docs skill">
  In the docs repo, run:

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  This improves page authoring and MDX component usage while you edit docs.
</Step>

<Step title="Use Claude Code for both product and docs work">
  Try prompts like:

  ```bash
  Search inspiration for strong skincare ads and draft a docs example from the result.
  ```
</Step>
</Steps>
