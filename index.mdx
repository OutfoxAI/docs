## What Outfox MCP does

Outfox MCP turns Outfox into a chat-native marketing workspace.

You can use it to:

- Search the inspiration library for winning ads
- Generate structured briefs from inspiration posts
- Inspect brands, products, and analysis jobs
- Add Meta analytics workflows to the same strategy loop

## Start with the loop

The best experience is not a single tool call.

It is the loop:

<Steps>
  <Step title="Search inspiration">
    Find examples worth discussing with `search_inspiration`.
  </Step>
  <Step title="Generate a brief">
    Convert a promising post into a structured direction with `generate_brief`.
  </Step>
  <Step title="Bring in performance context">
    Layer Meta analytics into the same conversation to see what patterns are
    actually winning.
  </Step>
</Steps>
