## What this tool family is for

The inspiration tool family helps you:

- search the ad library
- narrow results by platform, category, or media type
- inspect the best candidates
- turn a promising post into a brief for your brand

## Core tools

## Typical inputs

### `search_inspiration`

- `query`
- `platform`
- `category`
- `type`
- `liveOnly`
- `limit`

### `generate_brief`

- `brandId`
- `postId`

## Best prompts

```text
Search inspiration for direct-response pet ads on Facebook.
```

```text
Find five live supplement ads with strong hooks and tell me which hook patterns repeat.
```

```text
Generate a brief for my brand using the best inspiration post in that list.
```

## What a strong result looks like

A good inspiration result should give you:

- a post ID worth using in the next step
- a headline or hook pattern you can explain
- enough metadata to understand why the ad belongs in the set

A good generated brief should give you:

- a clear concept
- a hook
- a key message
- a CTA
- a visual direction
- a reason the source ad works

## Next step after the brief

Once you have a brief, ask the model to:

- make it more premium
- make it more direct-response
- make it fit your brand voice
- generate three testable variations
