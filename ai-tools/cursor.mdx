Set up Cursor for both Outfox MCP work and Mintlify docs editing.

## Prerequisites

- Cursor editor installed
- Access to an Outfox account with organization membership

## Setup

<Steps>
<Step title="Open the right repo">
  Open the app repo when you are working on MCP routes or landing pages.

  Open the docs repo when you are editing Mintlify pages.
</Step>

<Step title="Add the Outfox MCP server">
  In Cursor MCP settings, add:

  ```bash
  https://app.outfox.ai/api/mcp
  ```

  Complete the Clerk OAuth flow when prompted.
</Step>

<Step title="Install the Mintlify docs skill">
  In the docs repo terminal, run:

  ```bash
  npx skills add https://mintlify.com/docs
  ```
</Step>

<Step title="Preview docs locally">
  Run:

  ```bash
  mint dev --port 3333
  ```
</Step>
</Steps>
