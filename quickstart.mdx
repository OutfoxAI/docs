## Get started in four steps

Use this guide when you want Outfox MCP working fast.

### Step 1: Choose your MCP client

Outfox MCP is designed for clients that support remote MCP servers and OAuth.

- Claude
- Claude Code
- Cursor
- Other MCP-capable clients with OAuth support

### Step 2: Use the remote MCP endpoint

Use the Outfox endpoint:

```text
https://app.outfox.ai/api/mcp
```

If you are developing locally, use:

```text
http://localhost:3000/api/mcp
```

### Step 3: Complete Clerk OAuth

When your client connects, Clerk handles the sign-in flow and token exchange.

- Sign in with the account that has access to your Outfox organizations
- Approve the connection
- Let the client finish the OAuth handshake

<Tip>
  If your account has access to multiple organizations, start by asking the
  model to list them so you can pick the right workspace.
</Tip>

### Step 4: Run your first tool flow

Try one of these:

```text
List my organizations, then show me the brands I can access.
```

```text
Search inspiration for high-converting skincare video ads on Instagram.
```

```text
Generate a brief for my brand using the strongest inspiration post you find.
```

```text
Show me the top-performing creatives in this ad account for the last 14 days.
```
